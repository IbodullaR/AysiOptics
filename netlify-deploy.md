# Netlify orqali bepul HTTPS domain olish

## 1. Netlify.com ga ro'yxatdan o'ting
- https://netlify.com ga boring
- GitHub yoki email bilan ro'yxatdan o'ting

## 2. Static fayllarni tayyorlang
- shop.html, admin.html, login.html fayllarini alohida papkaga ko'chiring
- API URL'larni o'zgartiring

## 3. Deploy qiling
- Drag & drop qiling yoki GitHub'dan ulang
- Avtomatik HTTPS domain olasiz: https://random-name.netlify.app

## 4. Bot'da URL'ni yangilang
- Netlify domain'ni bot kodiga qo'ying
